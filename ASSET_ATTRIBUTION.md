# Asset attribution

The following uploaded free packs were visually inspected and curated for Phase 1.

## Tank pack

Source upload: `free-2d-battle-tank-game-assets.zip`

Used:

- Hulls from color sets A/B/C/D for player, enemy, ally and neutral faction variants.
- Weapons from color sets A/B/C/D for turret overlays.
- Explosion and muzzle flash sprites for combat animation.
- Shell sprites as reference art for projectile styling.

License note from pack: `https://craftpix.net/file-licenses/`

## Boat pack

Source upload: `craftpix-net-578218-free-top-down-military-boats-pixel-art.zip`

Used:

- Static boat sprites for identity/reference.
- Water-animation boat frames for the playable scout boat.

License note from pack: `https://craftpix.net/file-licenses/`

## RPG desert tileset

Source upload: `free-2d-rpg-desert-tileset.zip`

Used:

- Roads, buildings, decor, lake art and greenery where compatible with a top-down RTS presentation.

License note from pack: `https://craftpix.net/file-licenses/`

## Free Desert Top-Down Tileset

Source upload: `pack_free_desert_top_down.zip`

Used:

- Terrain tiles, rock edges, cacti, shrubs, dead trees, crates and desert props.
